# 🎓 Filtro de Alunos Aprovados

## 🎯 Propósito
Este é um projeto prático desenvolvido em JavaScript com o objetivo de treinar a manipulação de dados assíncronos e estruturas de repetição. O script lê um conjunto de dados no formato JSON, transforma essas informações em objetos manipuláveis e aplica uma lógica matemática para filtrar e exibir informações específicas com base no desempenho escolar.

## 📝 Enunciado
Com um arquivo JSON contendo uma lista de alunos em mãos, o programa deve executar as seguintes tarefas:

1. Transformar o texto JSON em objetos JavaScript nativos.
2. Desenvolver uma função que receba essa lista de objetos como parâmetro.
3. Calcular a média aritmética das notas de cada aluno.
4. Imprimir no console apenas os alunos que obtiveram **média maior ou igual a 7**, exibindo de forma formatada:
    - Nome
    - Média
    - Curso

## 🚀 Resultado Esperado
Ao executar o script (seja via ambiente Node.js no terminal ou integrado a uma página web), o sistema fará a leitura, o cálculo e exibirá a listagem filtrada no console, pulando linhas para facilitar a leitura.

A saída terá o seguinte formato:

```text
Alunos aprovados com nota 7 ou superior:

Aluno: Maria Joaquinha - Média: 8.17 - Curso: Sistemas para Internet
Aluno: João Ricardo - Média: 7.17 - Curso: Direito
Aluno: José Henrique - Média: 7 - Curso: Administração
...
```
